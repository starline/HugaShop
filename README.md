# HugaShop 🛒 Ecommerce CRM System

> **A powerful CRM platform for managing your online store, customers, and inventory.**

![Framework](https://img.shields.io/badge/framework-Symfony-blue?style=flat-square)
![ORM](https://img.shields.io/badge/ORM-Eloquent-orange?style=flat-square)
![Template](https://img.shields.io/badge/template-SMARTY-green?style=flat-square)

---

### ⚙️ Tech Stack:
- **Symfony** — robust PHP framework
- **Eloquent** — elegant ORM from Laravel
- **Smarty** — fast and flexible templating engine

---

### 💼 Features:
- 📚 **Content Management** — manage products, pages, filters, categories
- 👥 **Customer Relationship Management** — handle clients, segments, communication
- 🏬 **Warehouse Management** — track inventory, transfers, stock levels, and deliveries
- 💰 **Finance Management** — manage expenses, revenues, and transactions

---

> Ideal for online stores, distributors, and wholesale businesses.
